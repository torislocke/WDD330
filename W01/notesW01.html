Summary Notes

  • Pop up's irritate users - do not use.
  • Keep it simple and sleek.
  • Place Mobile menu's at bottom for ease of use.
  • Enable numbers keypad for mobile users when number input required.
  • Consider unmasking passwords.
  • State Menu - do not use hamburger icon
  • Speed Matters: SVG/Webfonts are lower bandwidth
  • Speed Matters: Optimize and minify CSS and JS
  • Speed Matters: Reduce image usage
  • Speed Matters: GZIP files to eliminate unnecessary data.
