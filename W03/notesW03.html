Summary Notes

  • Object values can be modified.
  • Any variable declared outside of a block is said to have global scope.
  • JavaScript Classes
  • Extends allows you to extend the parameters of one class into a new class, you may then add additional parameters unique to the new class. With Extends you inherit from the parent class Heavily used in REACT.
  • Mixins – use if you need to extend more than one class into another class. With Mixins you compose into a new class (not inherit) – not used much in REACT.
  • Map – takes an array and duplicates it as a new array in the same length.
  • Filter passes to a function and loops over every item in array and keeps the element requested in the filter if true.
