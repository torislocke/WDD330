Summary Notes - Chapter Ten (10)

JavaScript: Novice to Ninja, 2nd Edition

  • Three types of errors. Sytem error (problem with the system or external devices), Programmer error (program contains incorrect syntax or faulty logic), user error (user entered data the program is unable to handle).
  • ECMAScript 5 inoducd a strict mode which encourages a better quality of JavaScript. Add 'use strict'; as first line.
  • ES6 introduced JavaScript modules - self-contained pieces of code that are in strict mode by default.
  • Linting tools (JS Lint, JS Hint, ES Lint) help test the quality of JavaScript.
  • Feature detection is the recommended way to determine browser support.
  • Modernizr is a library to implement feature detection and Can I use.
  • Basic form of debugging is to use alert() to stop a programming from running until ok is clicked.
  • Browser Debugger to create breakpoint in code.
  • Testing Frameworks: Jest, created by Facebook.
